About mdbook

mdbook is a rust program that builds html documentation with markdown files.

The build process is simple for users. The only config needed is to make sure each page is denoted in your SUMMARY.md file. The pages in that file are the ones that will be built.

Options and arguments

There are different options that can be parsed.

       --hh, ----hheellpp

   SSUUBBCCOOMMMMAANNDDSS::
       build  Builds a book from its markdown files

       clean  Deletes a built book

       help   Prints this message or the help of the given subcommand(s)

       init   Creates the boilerplate structure and files for a new book

       serve  Serves a book  at  http://localhost:3000,  and  rebuilds  it  on
              changes

       test   Tests that a book's Rust code samples compile

       watch  Watches a book's files and rebuilds it on changes

       For  more  information  about a specific command, try `mdbook <command>
       ----hheellpp`   The   source   code   for    mdBook    is    available    at:
       https://github.com/rust-lang/mdBook
